Public Speaking Summit

PSS2023 is the second edition of the public speaking summit organized by the vocal array academy. This summit is one that is aimed at sharing value-oriented stories from world-class speakers.

Last year the theme was about the stage and speakers from across Nigeria were invited to speak. It was very impactful. This year, the spotlight would be on the attendees as we walk down the aisle to become the best version of ourselves in every way we can.

Role

UX/UI Designer designing the mobile app from conception to delivery.

Timeline

4 Weeks (June 2023)

Tools

  • Figma
  • Google suits

PROBLEM STATEMENT

Attendees sometimes find it hard keeping up with an event. It might be keeping up with the speakers or the not knowing about them.

Attendees might also want to find out more about the event like what is the event all about, the list of volunteers and sponsors.

SOLUTION

  • Design a mobile app for the attendees to enable them follow up as the event is going on.

EMPATHIZE PHASE

USER RESEARCH

I carried out research by finding inspirations from different event apps I could find from websites like Dribbble and Behance.

IDEA PHASE

BRAINSTORMING

All team members came together to hold a brainstorming session and came up with ideas. We came up with multiple options on how the UI for the speakers section would be, either in list or grid format.

We also came up with the idea of adding the whole order program of the program from start to finish.

DESIGN PHASE

STYLE GUIDE

I made sure to use the already established design system when designing.

WIREFRAME

I came up with the paper sketches first before designing the low-fi.

LOW FIDELITY

After making paper sketches and doing the necessary corrections, I started designing the Low-Fi.

HIGH FIDELITY

Using the style guide, I started work on the High-Fidelity and Prototype. I made sure to constantly communicate with mobile app developer to make any adjustments if necessary and for handing off my designs quickly after each iterations.

TEST PHASE

IMPLEMENTING FEEDBACK

One of the major feedback was to make the speaker section in list form.
